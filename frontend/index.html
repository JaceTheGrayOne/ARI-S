the Asset Reconfiguration and Integration System (ARI.S) facilitates direct manipulation of simulation architecture.

UWP Dumper

Extract encrypted files from Universal Windows Platform (UWP / Gamepass) applications.

Instructions:
1. Ensure your target UWP app is running
2. Click "Launch UWPDumper" below
3. In the console window, enter the Process ID (PID) of your UWP app
4. Files will be dumped to: %LOCALAPPDATA%\Packages\<PackageFamilyName>\TempState\DUMP
